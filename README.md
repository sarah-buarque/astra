# ASTRA - Ambiente de Saberes e Transmissão de Resultados Acadêmicos 

Nosso projeto tem como objetivo desenvolver um sistema web para facilitar a divulgação e participação em projetos no
IFRN. Ele surge da necessidade de tornar essas iniciativas mais visíveis, já que muitos alunos não sabem quais projetos
estão ativos no Campus. O sistema permitirá que projetos sejam cadastrados e que os estudantes possam visualizar e
acompanhar suas atividades e eventos.
